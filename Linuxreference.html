cd: go to home directory cd .. : go back to previous directory ls -al: pwd: see in which directory you are nano: create file in directory cd /(filename): goes into directory/folder man (command): manual (how to use) that command uname: print operating system name echo (string): write arguments to the standard output cp: copy mv: move cp: copy rm: remove file
Command Use Example
cd sends you back to your home directory, even if you're in another directory. A very useful command! If I was in my "scripts" directory, typing cd would send me home.
cd.. Takes you to your previous directory, not necessarily home. If I was in my "website" directory, and I typed cd.. it would take me to my "scripts" directory.
ls-al Gives you a complete list of things in a certain directory, all files. If I was was in my "scripts" directory, and I typed "ls -al", it would show me "myfirstbashscript" and "ping" and "output".
pwd When typed, it shows what directory you're in. If I typed in pwd right now, it displays /home/student4/website.
nano (file name) Creates a file in your specific directory, or it allows you to open the file if it already exists nano index.html opens the file index.html. nano apple would create a file called "apple"
